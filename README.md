## ✍🏻 제작 후기

- 본 포트폴리오는 **Next.js + TailwindCSS + Framer Motion** 기반으로 제작되었습니다.
- 디자인 통일성과 가독성을 중요하게 생각하며, 각 섹션은 재사용 가능한 컴포넌트로 구성했습니다.
- 개인 브랜딩을 위한 로고 및 애니메이션, 로딩 화면 등을 적용했습니다.
